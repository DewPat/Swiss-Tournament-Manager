# FIDE Chess Tournament Manager

A professional Swiss-system tournament management application designed for organizing chess events with accuracy, efficiency, and ease.

Built with real-world tournament rules in mind, this tool supports FIDE-style pairings, advanced tie-break calculations, and seamless data export for organizers.

---

## Features

- Swiss-System Pairings  
  Automatically generates fair pairings for each round.

- Buchholz Tie-Break System  
  Accurate player rankings using standard tie-break rules.

- Live Standings Table  
  Real-time updates as match results are entered.

- Excel Export Support  
  Export standings and results for reporting and sharing.

- Player Match History  
  Track each player's performance across rounds.

- Clean and Interactive UI  
  Simple and intuitive interface for smooth tournament management.

---

## Tech Stack

- Frontend: React.js  
- Logic: Custom Swiss pairing and tie-break algorithms  
- Utilities: Excel export handling
